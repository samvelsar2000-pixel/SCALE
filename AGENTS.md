# AGENTS

## Правила
- Не добавляйте бинарные файлы в репозиторий.
- Не коммитьте `node_modules/`, `dist/`, `.vite/` и другие build-артефакты.

## Команды проекта
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run deploy`

## Где менять контент
- Тексты секций и карточек: `src/data/*`
- Структура страниц: `src/pages/*`
- UI-компоненты: `src/components/*`
